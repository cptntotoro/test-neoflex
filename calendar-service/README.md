# Calendar Service

Сервис календаря.

Содержит праздничные дни за 2025 год, определяет выходные и рабочие дни.

## GET /days?startDate={startDate}&endDate={endDate}

Образец ответа:

```
{
    "startDate": "2025-01-01",
    "endDate": "2025-01-10",
    "holidays": [
        "2025-01-01",
        "2025-01-02",
        "2025-01-03",
        "2025-01-04",
        "2025-01-05",
        "2025-01-06",
        "2025-01-07"
    ],
    "nonWorkingDays": [],
    "workingDays": [
        "2025-01-08",
        "2025-01-09",
        "2025-01-10"
    ]
}
```
